The supplied tool's core dump behavior uses:
- 9600 8N2, DTR/DSR serial setup;
- SYST:RWL;
- DIAG:PEEK? -1,<0..255>,0;
- SYST:LOC;
- default output file caldump.bin.

This GUI reimplements the read-only protocol in Python with a graphical interface,
port enumeration, progress, validation, and no flash/write implementation.
