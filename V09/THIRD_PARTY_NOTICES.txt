The supplied tool's core dump behavior uses:
- 9600 8N2, DTR/DSR serial setup;
- SYST:RWL;
- DIAG:PEEK? -1,<0..255>,0;
- SYST:LOC;
- default output file caldump.bin.

This GUI reimplements the read-only dump protocol in Python with a graphical interface,
COM-port enumeration, progress, and checksum validation.

The v0.8 SCPI Command Console shares the same serial transport and *IDN? transaction
worker as the Dump Reader. It executes the exact non-comment text present in its
editable command script after a confirmation dialog. It does not implement HPAK flash mode.
